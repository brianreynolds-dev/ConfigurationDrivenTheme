# Release Notes

## 1.0.2
- Added a monitor for the debugger state, so theme changes apply while the Visual Studio debugger is running or paused.
- Added dropdown-based theme selection for fallback, debugging, and configuration mapping settings.

## 1.0.1
- Added a monitor for the active Visual Studio instance, so theme changes apply immediately when the user switches between multiple open instances with different active solutions and configurations.

## 1.0.0

- Added a dedicated extension icon and preview image for the VSIX listing.
- Improved the manifest description to explain the main benefit and configuration-driven behavior.
- Added packaged release notes and a getting started guide for Visual Studio Extension Manager users.
- Added configurable theme mappings for solution configurations.
- Added fallback theme restore behavior for unmapped configurations.
- Improved theme catalog handling and settings persistence.
